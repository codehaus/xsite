To run XSite:

- Define environment variable XSITE_HOME to point to the directory in which the binary distribution was extracted.
- Add XSITE_HOME/bin to system path

- xsite -Ssite -mcontent/sitemap.xml -stemplates/skin.html -Rresources -ooutput [-L<csv-of-l10n-codes>]

Note: On Windows you will need to replace / with \ in file paths.

XSite is a simple yet flexible site generation tool.

Features

  • A full separation between content and style
  • Support templatable skin
  • Easy configuration sitemap
  • Content expressed in plain HTML and not a tool-specific meta-language
  • Supports content, skins and even resources in multiple languages and locales.
  • Tool available in several forms, as an embeddable core API, as Ant task, as Maven Plugin, and as a standalone binary distribution that only requires a JRE.

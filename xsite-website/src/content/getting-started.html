Installing standalone XSite

Download XSite and unzip archive to your local filesystem, eg ~/xsite, which will thereafter be refered as XSITE_HOME. It is also useful to add XSITE_HOME/bin to your system path. On UNIX systems, ensure that you have the correct permissions to execute bin/xsite.sh.

Using XSite

The required inputs for XSite are

  • sitemap.xml: specifies the content of the website and the navigation. The content files refered to in the sitemap are assumed to be in the same directory as the sitemap.
  • skin.html: specifies the template and style used by XSite
  • output directory: specifies the directory where the generated files will be written to.
To generate site, simply execute (assuming xsite.sh is in system path, else use full path XSITE_HOME/bin/xsite.sh). 
xsite.sh [path-to-sitemap] [path-to-skin] [output-directory]
Executing script without arguments will also result in a usage message.

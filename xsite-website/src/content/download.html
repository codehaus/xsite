Using with Maven 2

Maven 2 users can easily add XSite as a dependency to their projects. Add the following repositories to the POM: 


		
			codehaus-m2-repository
			Codehaus Maven 2.x Repository
			http://repository.codehaus.org
		
		
			codehaus-m2-snapshot-repository
			Codehaus Maven 2.x Snapshot Repository
			http://snapshot.repository.codehaus.org
			
				true

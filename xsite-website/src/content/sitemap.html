Sitemap configuration

XSite can be easily configured using an XML-based sitemap: 


    

        About XSite
        index.html
        license.html
        download.html
        news.html
        changes.html
        versioning.html
    

    

        Using XSite
        faq.html
        list-user.html
        javadoc/index.html
        http://jira.codehaus.org/browse/XSITE
    

    

        Developing XSite
        list-dev.html
        team.html
        repository.html
    


]]>
Each page element corresponds to an HTML file in the content directory.

The sitemap filename is configurable but must be found in the content directory.
